Avoid ELBATIVENI.

Use your spell to move the rooms.

Attention, you can’t move a room you’ve already moved.

Good Luck …

![ScreenGif1](https://user-images.githubusercontent.com/43808018/161435585-67fcdea4-e2d4-4caa-8d7e-8e8cdf701425.gif)

Game created in 48 hours for the Compo of the Ludum Dare 50 : Delay the inevitable

Controls :

Point and click to move

Select rooms to swap
